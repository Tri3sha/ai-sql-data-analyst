# 🤖 AI SQL Data Analyst Agent

This project converts natural language queries into SQL and analyzes CSV data.

## Features
- Upload CSV
- Ask questions in plain English
- Generates SQL queries
- Displays results

## Tech Stack
- Streamlit
- LangChain
- Groq LLM
- SQLite
- Pandas

## Run Locally
```bash
streamlit run app.py
